Перевод: Что такое Shadow DOM

Пару недель назад был опубликован пост про то, что из себя представляет DOM.

Чтобы освежить понятие в памяти, DOM (Объектное представление документа) это представление HTML кода в виде объекта, который используется браузером для определения того, что должно быть отрисовано в окне. Так же DOM используется в JavaScript для редактирования контента и структуры страницы, а так же для стилизации элементов.
